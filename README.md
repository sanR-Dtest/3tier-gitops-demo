# 3-Tier GitOps Demo

A demonstration of a three-tier application deployment using Kubernetes, Helm, and ArgoCD with GitOps principles.

## Overview

This repository contains:
- **Frontend**: User-facing web application
- **Backend**: REST API service
- **Database**: MySQL persistence layer

## Structure

- `charts/` - Helm charts for deploying the application
- `argocd/` - ArgoCD application definitions
- `environments/` - Environment-specific configurations (dev, qa, prod)
- `diagrams/` - Architecture diagrams
